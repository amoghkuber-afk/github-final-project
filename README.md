# Simple Interest Calculator

This project contains a simple Bash script that calculates **simple interest** based on user input.

## Description
The Simple Interest Calculator computes the interest earned on a principal amount over a certain time period.  
The script asks the user to enter the **principal amount, rate of interest, and time period**, then calculates the simple interest.

## Formula Used

Simple Interest = (Principal × Rate × Time) / 100

Where:
- **Principal (P)** = Initial amount of money
- **Rate (R)** = Annual interest rate
- **Time (T)** = Time period in years

## How to Use

1. Run the bash script:
 
2. Enter:
- Principal amount
- Rate of interest
- Time period

3. The script will display the **simple interest**.

## Example

Input:
Principal = 1000  
Rate = 5  
Time = 2  

Output:
Simple Interest = 100
